> vp dlx --help # should show help message
Execute a package binary without installing it as a dependency

Usage: vp dlx [OPTIONS] <ARGS>...

Arguments:
  <ARGS>...  Package to execute (with optional @version) and arguments

Options:
  -p, --package <NAME>  Package(s) to install before running the command (can be used multiple times)
  -c, --shell-mode      Execute the command within a shell environment
  -s, --silent          Suppress all output except the executed command's output
  -h, --help            Print help
